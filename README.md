## origin-website

Website for the origin art collective

## Running the project

For the first time, run

```
  npm i
```

Start the project with this and browse at localhost:8080

```
  npm start
```

Build the project with (built files are in /docs)

```
  npm run build
```

Run tests with:

```
  npm run test
```

To see test coverage:

```
  npx jest --coverage
```
